================================================================================
VOCABULARY SPECIFICATION
================================================================================

Total Size: ~270 tokens

Special Tokens (4):
  0: <PAD>  - Padding
  1: <UNK>  - Unknown character
  2: <BOS>  - Beginning of sequence
  3: <EOS>  - End of sequence

Devanagari (160 characters):
  U+0900 to U+097F - All Devanagari script characters
  Includes: vowels, consonants, matras, nukta, chandrabindu, etc.

ASCII Punctuation & Space (6):
  Space, period, comma, exclamation, question, colon, hyphen

Total vocabulary is fixed and deterministic - same across all runs.

================================================================================
DATA FORMAT SPECIFICATIONS
================================================================================

TRANSCRIPT FILE (input to prepare_dataset.py):
  Format: TSV with 4 columns
  audio_filename.wav<TAB>hindi_text<TAB>speaker_id<TAB>gender
  Encoding: UTF-8
  Example:
    sample_001.wav  नमस्ते दुनिया  speaker_001  M
    sample_002.wav  यह परीक्षण है  speaker_002  F

MANIFEST FILE (output from prepare_dataset.py, input to datasets):
  Format: JSON
  Content: Array of objects with keys:
    - audio_path (string): Path to audio file
    - text (string): Original Hindi text
    - speaker_id (string): Speaker identifier
    - duration (float): Audio duration in seconds
    - gender (string): M/F or other
  Encoding: UTF-8

LATENT FILE (output from precompute_latents.py):
  Format: NumPy .npy file
  Content: 1D array of latent values (float32)
  Naming: {audio_stem}.npy
  Path: precomputed_latents/{audio_stem}.npy

CONFIGURATION PARAMETERS
================================================================================

TEXT PROCESSOR:
  - Language: "hi" (fixed for Hindi)
  - Special tokens: Fixed (PAD, UNK, BOS, EOS)
  - Vocabulary: Fixed (270 tokens)

AUDIO PROCESSOR:
  - Sample rate: 44100 Hz (configurable)
  - N_FFT: 2048 (configurable)
  - Hop length: 512 (configurable)
  - N_mels: 228 (configurable)
  - Frequency range: 55 Hz to 22050 Hz (configurable)
  - Window: Hann (fixed)

DATASET:
  - Max text length: 256 characters (configurable)
  - Normalize text: True (configurable)
  - Load audio: True/False (configurable)

PREPARATION:
  - Min duration: 0.5s (configurable)
  - Max duration: 30s (configurable)
  - Max silence ratio: 0.5 (configurable)
  - Train/val/test split: 0.9/0.05/0.05 (configurable)
